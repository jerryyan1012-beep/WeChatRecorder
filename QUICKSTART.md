# 快速开始指南

## Windows 用户

### 最简单的方式（推荐）

1. **下载项目**
   - 下载 ZIP 压缩包并解压
   - 或使用 `git clone`

2. **双击运行**
   - 双击 `run_windows.bat`
   - 首次运行会自动安装依赖（需要几分钟）
   - 之后直接使用

### 遇到问题？

运行环境检查：
```cmd
python check_windows.py
```

或运行自动设置：
```cmd
python setup_windows.py
```

## 系统要求

- Windows 10/11 (64位)
- 至少 4GB RAM
- 500MB 可用磁盘空间
- 微信桌面版
